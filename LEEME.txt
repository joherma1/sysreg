Ejecucion:
	Utilizar RXTX 2.2-
Nota IMPORTANTE:
	Debido a que arduinoIDE-0021 utiliza la version estable si, comprobado en MAC, tenemos instalada la version 2.2 no funcionara correctamente, cierres inesperados. Para trabajar con arduino IDE deberemos borrar las libreriras RXTX 2.2 y para ejecutar de nuevo la aplicación volveremos a copiarlas en /Library/Java/Extensions

Mac OS X 10.6
	RXTX 2.1 Stabel
		Instalar RXTX 2.1-7-bins-r2
		java -d32 -jar SysAdmin.jar
	RXT 2.2 Pre
		Instalar RXTX 2.2pr2 segun el fichero instalación Mac OS X
		Doble click aplicacion => java -jar SysAdmin.jar
	

Ubuntu 10.04
	x32
		Instalar RXTX 2.1-7-bins-r2 siguiendo el fichero instalacion Ubuntu
		ó
		Instalar RXTX 2.2 Pre
		sudo cp rxtx-2.2pre2-bins/RXTXcomm.jar /usr/lib/jvm/java-6-openjdk/jre/lib/ext/
		sudo chmod 755 /usr/lib/jvm/java-6-openjdk/jre/lib/ext/RXTXcomm.jar
		sudo cp rxtx-2.2pre2-bins/i686-pc-linux-gnu/* /usr/lib/jvm/java-6-openjdk/jre/bin/
		chmod 755 /usr/lib/jvm/java-6-openjdk/jre/bin/librxtx*

		Abrir .jar con OpenJDK
	
	x64
		igual que en x32 con la carpeta x86_64-unknown-linux-gnu
	
Windows XP
	x32
		Instalar RXTX 2.2
			Copiar .dll en java/jre/bin
			Copiar .jar en java/jre/lib/ext
		Cambiar el puerto del arduino a COM11
		Doble click archivo
	
Windows 7 
	x64
		Instalar java jdk 1.6.0_21
		Instalar RXTX 2.2pre2 version 64 bits
			copiar como dice el fichero Instalacion XP
